FUNDAMENTALS UNICODE
====================

1. Description

       FUNDAMENTALS UNICODE is a Delphi/FreePascal unit that
       implements routines and classes for unicode string handling.


2. Requirements

       Borland Delphi 5-2007
       FreePascal 2
